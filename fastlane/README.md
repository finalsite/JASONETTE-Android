# Installation

Make sure you have the latest version of the Xcode command line tools installed:

```
xcode-select --install
```

Install _fastlane_ using
```
[sudo] gem install fastlane -NV
```
or alternatively using `brew cask install fastlane`

# Available Actions
## Android
### android prepare_for_deploy
```
fastlane android prepare_for_deploy
```
Create APK/AAB files and generate screenshots
### android deploy
```
fastlane android deploy
```
Deploy a new version to the Google Play Store, deploys to beta environment by default, adding the production flag will allow production deployment
